Detailed Description

Simple example program for the Z51 driver.

Author:
ub

Causes the Z51 to set its outputs either on a constant current/voltage or output a sawtooth wave on both outputs. See usage info.

Required: libraries: mdis_api, usr_oss

Preprocessor Switches:
(none)
